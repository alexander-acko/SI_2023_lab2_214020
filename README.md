### Цикломатска комплексност - 11
Според CFG има вкупно N = 28 јазли и E = 37 врски, па цикломатската комплексност ќе се пресмета според следната формула: 
Cyclomate complexity = E - N + 2 = 37 - 28 + 2 = 11
